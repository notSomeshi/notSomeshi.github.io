// 怎么读这张图

  • 柱长 = 跑完整套评测的总成本;右侧四列依次为总成本、平均每道题的成本、每 1 分智能指数的成本(总成本 ÷ 指数,绿色 = 低于 $10/分)、AA 智能指数。红柱为成本最高的三名。
  • 单价便宜 ≠ 总价便宜。推理型模型"想得多":Kimi K3 的每 token 单价只有 Opus 4.8 的约 1/7,但整卷烧掉 $2,710——它平均每题产出的推理 token 是非推理模型的数倍。看 API 标价选模型会踩这个坑,"同卷总价"才是真实成本。
  • 顶部智能的边际价格陡增。从 GLM-5.2 的 48.3 分爬到 Opus 4.8 的 61.4 分,每分成本从 $5.6 涨到 $62.7,涨了 11 倍——"最后 13 分"是全场最贵的商品。这正是「批量任务用开源、攻坚任务上旗舰」分层的定量依据(参见本站《OpenRouter 上的权力转移》)。
  • 甜点区在哪:50 分以上阵营里,DeepSeek-V4($7.8/分)和 Qwen3.7 Max($10.0/分)是单位智能最便宜的选择;旗舰档内部,Fable 5($40.4/分)比 Opus 4.8($62.7/分)便宜三分之一还多 1.5 分,是"旗舰里的性价比之选"。

口径:Artificial Analysis「Cost to Run Intelligence Index」——按各模型 API 的输入/缓存/推理/输出 token 实际计费汇总。实测锚点:Kimi K3 全套 $2,709.75;Claude Opus 4.8 为最贵档,约 $1.78/题;「$/题」按全卷约 2,160 题折算。其余为按各模型 API 定价与耗token量折算的估算值,数据快照 2026-07-18,以 AA 官网实时数据为准。

※ 极限用量为按各家官方额度窗口满负荷折算的数量级估算,不同任务类型(补全/对话/Agent)折算差异极大,仅供比较参考,以官方说明为准。会员折算价 = 订阅月费 ÷ 估算极限用量,即"把额度用满时每百万 token 的实际成本"——用得越少,实际单价越高;与右侧 API 标价对照,可以直观看出会员在什么用量下开始划算。国产平台的 API 价按当期汇率近似换算为美元。
